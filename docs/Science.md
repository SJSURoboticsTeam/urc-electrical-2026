# Science

![Top of PCB](outputs/POE/top.png)

## Features

- voltage regulation for 40V, 24V, 5V, and 3.3V
- Hall effect sensors
- color sensors
- 360 and 270 degree servos
- PWM
- I^2 C communication
- separate battery for fan
- lasors
